# Steps - 个人日程管理应用

一个简洁美观的个人日程管理应用，帮助你记录每日任务、心情和心得。

## ✨ 功能特性

- 📅 **日历视图** - 直观的日历界面，轻松浏览和选择日期
- ✅ **任务管理** - 添加、完成、删除每日任务
- 🎯 **状态标记** - 标记日期状态（计划中、已完成、未完成）
- 😊 **心情记录** - 记录每日心情（😊 😐 😢）
- 📝 **笔记功能** - 记录每日心得和想法
- 📊 **数据统计** - 显示每月打卡天数统计
- 💾 **本地存储** - 数据保存在浏览器本地，无需服务器
- 📱 **响应式设计** - 完美适配手机和电脑端

## 🚀 快速开始

### 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 或者将项目部署到任何静态网站托管服务

### 本地运行

```bash
# 使用 Python 简单服务器
python -m http.server 8000

# 或使用 Node.js
npx http-server
```

然后在浏览器中访问 `http://localhost:8000`

## 📖 使用说明

### 日历操作
- 点击左右箭头切换月份
- 点击日历中的日期查看或编辑该日期的数据
- 当前日期会以高亮显示
- 已记录的日期会显示状态点：
  - 🟢 绿色：已完成
  - 🔴 红色：未完成
  - 🟡 黄色：计划中

### 任务管理
- 点击右上角的 `+` 按钮添加新任务
- 点击任务前的圆圈标记任务为已完成
- 点击任务文本可直接编辑
- 点击 `×` 删除任务

### 状态和心情
- 使用下拉菜单选择日期状态
- 点击表情符号记录今日心情
- 在文本框中记录每日心得

### 数据保存
- 所有数据自动保存到浏览器本地存储
- 无需担心数据丢失，刷新页面后数据依然存在

## 🛠️ 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式和响应式布局
- **JavaScript (ES6+)** - 交互逻辑
- **LocalStorage** - 数据持久化

## 📱 PWA 支持

应用支持添加到手机主屏幕：
- iOS: 使用 Safari 浏览器，点击分享按钮选择"添加到主屏幕"
- Android: 使用 Chrome 浏览器，点击菜单选择"添加到主屏幕"

## 🎨 界面特点

- 现代化的卡片式设计
- 清新的配色方案
- 流畅的交互动画
- 直观的用户体验

## 📝 数据格式

数据以 JSON 格式存储在浏览器的 LocalStorage 中，每个日期的数据包含：
- `tasks`: 任务列表
- `note`: 笔记内容
- `status`: 状态（planned/completed/missed）
- `mood`: 心情表情

## 🔒 隐私说明

- 所有数据仅存储在本地浏览器中
- 不会上传到任何服务器
- 完全离线可用
- 清除浏览器数据会删除所有记录
